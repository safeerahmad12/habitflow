# HabitFlow

HabitFlow is a full-stack habit tracking SaaS application that helps users build routines, maintain streaks, and analyze productivity through visual insights.

## Live Demo
https://habitflow-eosin.vercel.app

## Features

- Habit creation and management
- Daily and weekly habit tracking
- Habit streak tracking
- Weekly analytics charts
- Activity heatmap visualization
- Secure authentication system
- Google OAuth login
- OTP email verification
- Password reset via email
- Guest login option
- Profile settings panel

## Tech Stack

### Frontend
- React (Vite)
- Recharts
- Modern responsive UI

### Backend
- FastAPI
- SQLAlchemy
- JWT Authentication

### Database
- PostgreSQL / SQLite

### Deployment
- Frontend: Vercel
- Backend: Render
